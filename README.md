# Roughfts Portfolio

A modern, minimalist portfolio showcasing Roughfts' development projects and skills.

## Technologies Used
- React
- TypeScript
- Styled Components
- Framer Motion
- React Router DOM

## Features
- Responsive design
- Smooth animations
- Project showcase
- Contact information

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm start`
